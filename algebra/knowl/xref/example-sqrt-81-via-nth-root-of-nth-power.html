Example 1.5.63.

Find \(\sqrt{81}\text{.}\)
🔗
Solution.
Inside the radical is \(81\text{,}\) which we can replace with \(9^2\text{,}\) since \(9^2\) and \(81\) are equal. That is,
\begin{equation*} \sqrt{81} = \sqrt[2]{81} = \sqrt[2]{9^2}. \end{equation*}
Note that in the above, we still had to write the second root: we have not processed the root here yet (we have only replaced \(81\) with something that is equal to \(81\)). Now, we can use the formula \(\sqrt[n]{a^n} = a\) to process the radical: \(\sqrt[2]{9^2} = 9\text{.}\) Presenting all the work with equal signs, we have
\begin{equation*} \sqrt{81} = \sqrt[2]{81} = \sqrt[2]{9^2} = 9. \end{equation*}
