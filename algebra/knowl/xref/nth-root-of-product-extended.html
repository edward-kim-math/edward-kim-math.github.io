Note 1.5.68.

There are extended versions of these formulas for three factors, namely \(\sqrt[n]{abc} = \sqrt[n]{a} \sqrt[n]{b} \sqrt[n]{c}\) and we show below what this looks like for square roots and cube roots:
  • \(\displaystyle \sqrt{abc} = \sqrt{a} \sqrt{b} \sqrt{c}\)
    🔗
    🔗
  • \(\displaystyle \sqrt[3]{abc} = \sqrt[3]{a} \sqrt[3]{b} \sqrt[3]{c}\)
    🔗
    🔗
but there’s an even more extended version for four factors, namely \(\sqrt[n]{abcd} = \sqrt[n]{a} \sqrt[n]{b} \sqrt[n]{c} \sqrt[n]{d}\text{,}\) and for square roots and cube roots, this means:
  • \(\displaystyle \sqrt{abcd} = \sqrt{a} \sqrt{b} \sqrt{c} \sqrt{d}\)
    🔗
    🔗
  • \(\displaystyle \sqrt[3]{abcd} = \sqrt[3]{a} \sqrt[3]{b} \sqrt[3]{c} \sqrt[3]{d}\)
    🔗
    🔗
and so on: there are extended versions of this for any number of factors. We just need to ensure that all radicals are of the same kind (that is, we have the same little number on the outside of all radicals), and that we have multiplication everywhere (instead of addition).
