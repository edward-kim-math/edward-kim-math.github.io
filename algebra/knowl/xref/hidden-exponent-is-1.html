Note 1.5.2.

In the desire to make sure we have really said everything (no math secrets here!), let’s also say that \(a^1 = a\text{.}\) It makes sense that if \(a^3 = a \cdot a \cdot a\) and \(a^2 = a \cdot a\text{,}\) then to follow the pattern \(a^1\) must equal \(a\text{.}\) So \(u^1=u\) and \(8^1=8\) and \((x+7)^1 = x+7\text{,}\) for example.
🔗
There are times that we’ll need to simplify an expression like \(p^1\) as \(p\text{.}\) There will actually be other times we need to go in reverse, turning \(p\) into \(p^1\text{.}\) Informally, we can think of it this way: whenever we see an expression, if there is no exponent explicitly written in, we can always write in an exponent of \(1\) explicitly. I like to call this the “hidden exponent of \(1\)”, analogous to the “hidden denominator of \(1\)” that we discussed in Note 1.2.4.
