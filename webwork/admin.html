Reset the password someone uses to access WeBWorK on the web

User (typically instructor) forgot their password.
  • From the bash shell, run {% highlight latex %} newpassword [course name] [username] [new password] {% endhighlight %} For example, {% highlight latex %} newpassword MTH456-Kim ekim 12345678 {% endhighlight %}

Unhiding courses

From Course Administration, click on hide courses. Sort courses alphabetically. Select the appropriate course. Then select the button to hide or unhide the course.

Adding a symbolic link to the Open Problems Library

From the bash shell, run {% highlight latex %} cd /opt/webwork/courses/[course name]/templates ln -s /opt/webwork/libraries/webwork-open-problem-library/OpenProblemLibrary Library {% endhighlight %} For example, {% highlight latex %} cd /opt/webwork/courses/MTH456-Kim/templates ln -s /opt/webwork/libraries/webwork-open-problem-library/OpenProblemLibrary Library {% endhighlight %}
